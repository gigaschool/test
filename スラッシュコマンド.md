## 補足

重複や別名は以下です。

| 同義・別名                        | 意味              |
| ---------------------------- | --------------- |
| `/allow-all` = `/yolo`       | すべての権限を有効化      |
| `/clear` = `/new`            | 会話履歴を消去して整理     |
| `/exit` = `/quit`            | CLI終了           |
| `/streamer-mode` = `/on-air` | 配信向け表示モード切替     |
| `/cwd` と `/cd`               | 前者は表示寄り、後者は移動寄り |

この一覧で見える本質は、Copilot CLI は単なるチャットUIではなく、次の5系統を持っています。

1. **権限管理**
   `/add-dir` `/list-dirs` `/allow-all` `/reset-allowed-tools`

2. **開発支援**
   `/diff` `/plan` `/review` `/delegate` `/research`

3. **拡張機能管理**
   `/skills` `/plugin` `/mcp` `/lsp`

4. **セッション管理**
   `/session` `/resume` `/rename` `/restart` `/compact`

5. **環境・表示管理**
   `/theme` `/usage` `/model` `/user` `/terminal-setup`

必要なら次に、これをさらに整理して
**「初心者が最初に覚えるべき10コマンド」**
**「実務でよく使う順」**
の2表に再構成します。
